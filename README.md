# BulaFácil

App para verificar interações entre medicamentos.

## Arquivos
- main.py → Backend (Python/FastAPI)
- medicamentos.json → Base de medicamentos
- requirements.txt → Dependências Python
- index.html → Interface do usuário

## Deploy no Railway
1. Criar conta em railway.app
2. "New Project" → "Deploy from GitHub"
3. Conectar este repositório
4. Adicionar variável: GEMINI_API_KEY
5. Aguardar deploy automático

## URL do frontend
Após deploy, copie a URL do Railway e cole no index.html (linha ~270)
